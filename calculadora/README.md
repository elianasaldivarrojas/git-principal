# Calculadora - Operaciones básicas

Archivos:

- `index.html` - Interfaz de la calculadora.
- `styles.css` - Estilos.
- `script.js` - Lógica de la calculadora.

Instrucciones rápidas:

1. Abrir `index.html` en un navegador (doble clic o desde la terminal).
2. La calculadora soporta teclado: números, `+ - * /`, `Enter` para = y `Backspace`.
